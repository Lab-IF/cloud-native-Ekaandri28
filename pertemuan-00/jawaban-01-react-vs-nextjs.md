## 📝 Kuis Singkat

### 1. Next.js adalah...

**Jawaban:** _(Next.js adalah framework berbasis React yang digunakan untuk membangun aplikasi web modern dengan performa tinggi dan struktur yang lebih rapi dibanding React biasa.)_

### 2. Mana yang TIDAK perlu di-install manual di Next.js?

**Jawaban:** _(Di Next.js, yang TIDAK perlu di-install manual adalah routing (react-router) karena sudah tersedia otomatis dari struktur folder.)_

### 3. Saat install Next.js, agar menggunakan JavaScript (bukan TypeScript), kita harus pilih...

**Jawaban:** _(No)_

### 4. Apakah komponen React bisa dipakai di Next.js?

**Jawaban:** _(Next.js dibangun di atas React, jadi semua komponen React bisa langsung digunakan di Next.js tanpa perubahan besar.)_

### 5. Sebutkan minimal 3 fitur yang Next.js berikan di atas React biasa!

**Jawaban:** _(Routing otomatis → tidak perlu install react-router
Server-Side Rendering (SSR) & Static Site Generation (SSG) → untuk performa dan SEO lebih baik
API Routes (backend bawaan) → bisa buat API tanpa framework tambahan)_
